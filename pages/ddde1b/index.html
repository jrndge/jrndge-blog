如果成为苹果独立开发者

# 都在说苹果上架应用有人内购就会有收入\

是的,要么通过支付,要么通过内购,

如果是虚拟商品的话,就一定要走苹果支付,如果是实体商品的话,可以走三方支付,比如外卖等.

# 如果成为一名苹果开发者

直接进行申请就可以了.

# 怎么申请苹果账号

# 申请条件

  1. 年满18周岁
  2. 有效的身份证件
  3. 可以使用的信用卡(用于支付年费)
  4. 有效的电子邮箱

# 申请步骤

  1. 创建Apple ID
  2. 登录Apple Developer网站
  3. 同意开发者协议
  4. 填写个人信息
  5. 完成付款($99)
  6. 等待审核

# 必须要我有一个苹果设备才可以

https://developer.apple.com/enroll/app (opens new window)

在 iPhone、iPad 或 Mac 上打开 Apple Developer App。选择“账户”标签页并登入，继续下一步操作。

# 没有苹果电脑可以开发苹果应用吗?

可以开发

  1. Windows可以使用vm搭建本地的mcos环境进行开发
  2. 使用Flutter语言进行开发.

开发语言选择三种进行对比一下.

官方提供了两种,都只是支持mac系统

另外一种支持Windows开发.无需mac环境

xcode,Swift,还有Flutter

# 个人开发者账号和企业开发者账号的区别

区别1:费用

个人开发者$99/年

企业开发者$299/年

实际上普通的企业开发者发布的应用也只能企业内部员工使用,不能使用苹果的分发功能,也就是在商店里面搜不到吧(我是这样人为的,不知道对不对).

个人开发者也同样可以接入支付系统.只不过更多的是苹果的支付.已经很不错了.支付宝微信,个人支付都不能接的.

但是苹果那边要抽30%,好像说年收入低于$100w可以申请降低至15%.

不知道申请的难度如何?

不知道有没有人申请成功.
